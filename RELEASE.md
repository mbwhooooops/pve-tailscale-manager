# Release Notes

## v0.1.0

Initial packaged release of the local Proxmox VE Tailscale integration.

### Added

- Debian package structure for `pve-tailscale-integration`
- Custom Proxmox API module: `PVE::API2::Nodes::Tailscale`
- Automatic patch helper for `Nodes.pm` and `pvemanagerlib.js`
- Proxmox node UI panel for Tailscale configuration and peer visibility
- Package install hook that restarts `pvedaemon` and `pveproxy`

### Fixed

- Invalid `tailscale up` invocation caused by an empty `--exit-node=` argument being generated by the backend

### Operational Notes

- The package is intended for local deployment and maintenance
- Reinstallation may be required after future `pve-manager` upgrades
- Patch logic is idempotent but depends on current Proxmox file structure
